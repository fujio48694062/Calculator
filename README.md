# 電卓（VBA）
右クリックによって数値を入力し、四則演算を行う電卓です。
![電卓](https://user-images.githubusercontent.com/84171334/188270894-ca6e8d0e-32c6-496b-b80f-74834ba90727.jpg)

## 実行内容
- 四則演算（加算・減算・乗算・除算）
- 算術演算子（+, -, *, /, (, )）
- AC：入力された数値を全て削除
- BS：直前に入力した1桁を削除
